# E-commerce Website

A responsive front-end e-commerce website built with **HTML5**, **CSS3**, and **JavaScript**. This project showcases modern web design principles with a focus on user experience and clean code architecture.

## 🎯 Features

- **Responsive Design** - Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices
- **Product Showcase** - Display of products with images, descriptions, and pricing
- **User-Friendly Interface** - Intuitive navigation and clean UI design
- **Semantic HTML** - Well-structured HTML for better accessibility and SEO
- **Modern CSS** - Professional styling with CSS3 features for an attractive visual presentation
- **Interactive Elements** - JavaScript-powered interactions for enhanced user engagement

## 💻 Tech Stack

- **HTML5** - Semantic markup and structure (63.6%)
- **CSS3** - Responsive styling and layout (32.9%)
- **JavaScript** - Interactive functionality (3.5%)
